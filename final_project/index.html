Educational Attainment by Race/Ethnicity and Gender

The following data comes from the United States Census Bureau. They have compiled multiple datasets depicting the educational attainment of those 18 years and older in the United States. Learn about the disparities of educational attainment compared between race and ethnicity, as well as gender, below. Note: Notice how the y-axis values change when different genders are selected.

Explanation

The y-axis represents the number of graduates who have attained a certain level of education While the x-axis represents the level of educational attainment by grade level, and each color on the line chart depicts a race or ethnicity, On the far right side of this visualization is a filtering system for the audience to choose which gender(s) to display and compare. By representing educational attainment through a line chart, the audience can easily view the trend of educational attainment between each group and determine which races achieve higher or lower education. The user may select one gender at a time via radio buttons to view the races by gender.

Analysis

While viewing this overall line chart above, we can see that there are some large discrepancies. When any gender is selected, the number of white graduates far surpasses the number for any other race/ethnicity. When we switch viewing between genders, the number of graduates slightly lessens from male to female. As the y-axis and number of graduates change with gender, we see the overall pattern of the chart stay the same with the number of white graduates highly exceeding the number of Asian, Black, and Hispanic graduates.
